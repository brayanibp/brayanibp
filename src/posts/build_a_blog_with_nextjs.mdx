## Ok, so what is Next.js?

Next.js is a powerful tool to develop dynamic web applications using javascript or typescript based in React.js library. It's mission is to make SPA and dynamic page applications easy and fast to develop but for sure functional.
Some people could say is pretty similar to PHP with a just drop in a jsx file it would be a page in your site. 

## But how does it work?

Next.js within its mission has a built in router functionality to give us mind peace, with a modular approach using <InlineHighlighter>{`src/app`}</InlineHighlighter> folder structure where all files dropped in will work as a page in your web application.
